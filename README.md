I spend a lot of time thinking before I build.

I work with Python, data analysis, and data science, but most of my work starts earlier than code — with questions, patterns, and things that don’t fully make sense yet. I like staying there for a while.

I look at data the way I look at people: shaped by context, bias, structure, and quiet assumptions. Accuracy matters, but understanding matters more.

I’m learning as I go. Building slowly. Letting things take the time they need.
